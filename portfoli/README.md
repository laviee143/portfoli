# Student Portfolio Profile Card

A minimal and modern profile card created for a **Software Engineering student**.
This project focuses on clean layout, readability, and responsiveness without using any frameworks.

## Overview

The profile card displays personal information, a short bio, social links, and a contact form.
It is suitable for beginner portfolios, university assignments, or personal landing pages.

## Main Highlights

* Single-column layout that works on all screen sizes
* Navy blue theme with a professional look
* Simple contact form with basic validation
* No external JavaScript libraries
* Easy to customize and extend
* Responsive UI

## Built With

* HTML5 for structure
* CSS3 for layout and styling
* Vanilla JavaScript for form handling
* Font Awesome for icons

## Folder Structure

```
.
├── index.html      # Main HTML file
├── style.css       # Styles and layout
├── script.js       # Form validation logic
├── image.png         # Profile image
└── README.md       # Project documentation
```

## Usage

1. Download the project files
2. Place all files in the same directory
3. Open `index.html` in any modern web browser

No installation or build steps are required.

## Customization Guide

* Update personal details inside `index.html`
* Replace `image.png` with your own image
* Modify colors and spacing in `style.css`
* Enhance the contact form logic in `script.js`

## Intended Audience

This project is ideal for:

* Software Engineering students
* Beginners learning HTML, CSS, and JavaScript
* Personal portfolio starters
